**A consolidated monorepo containing personal robotics experiments and learning implementations across core Robotics Software Engineering domains.**

Rather than maintaining disconnected micro-repositories, this sandbox provides a single, structured space to track my iterative skill development — from hardware abstraction layers in C++ and Python, through 3D simulation with Gazebo, to deep learning perception models.

## 📂 Repository Structure

| Domain | Directory | Key Project | Tech |
|--------|-----------|-------------|------|
| **C++ for Robotics** | [`/cpp_course`](./cpp_course) | `robot_cpp` — HAL with OOP, smart pointers, sensor abstraction | `C++17`, `CMake` |
| **Simulation** | [`/GAZEBO`](./GAZEBO) | `gazebo_tutorial` — SDF robot model (FWS robot) in Gazebo Harmonic | `Gazebo Harmonic`, `SDF`, `ROS 2` |
